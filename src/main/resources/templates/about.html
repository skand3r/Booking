About This Project

This is a Spring project utilizing the Spring Framework and Spring Boot to develop a booking website. The platform allows users to search for hotels, explore available options, and manage their bookings. While the project is functional, it is a work in progress, with new features being added periodically. Currently, the primary focus is on hotel bookings. However, the project is designed with scalability in mind and could be extended to support other types of bookings in the future, such as plane tickets, concerts, and more. The project's features, documentation, and source code are available on my GitHub account. You can find the relevant links below.

Note: This project is for educational purposes only.
